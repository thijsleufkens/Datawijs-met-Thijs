# Datawijs met Thijs

Een verzameling experimentele appjes rond besluitvorming in de Nederlandse maakindustrie. Gebouwd door [Thijs Leufkens](https://datawijs.nl) — onafhankelijk BI-consultant in Venlo, gespecialiseerd in het vertalen van data naar bruikbare stuurinformatie voor maakbedrijven.

## Wat is dit

Dashboards tonen *de stand*. Maar besluitvorming bestaat uit veel meer: wie heeft het gezien, wat dachten ze ervan, wat is besloten, met welke verwachting, en wat kwam eruit. Die context leeft nu nergens vastgelegd.

Deze repo verzamelt kleine appjes die proberen die context te vangen — als experiment, als gesprekstof, en als bewijsmateriaal dat ik niet alleen praat over data maar er ook dingen mee maak.

Het overkoepelende werknaam-concept is *Decision OS*: een schil rond Power BI-dashboards die de "zachte" kant van besluitvorming meeneemt. Het is geen product, geen startup, geen roadmap. Het is een speeltuin met richting.

## Waarom dit publiek is

Omdat het bewijsmateriaal is voor wat ik doe. Ik werk in de spagaat tussen techniek en bedrijfskunde, en ik geloof dat de beste manier om dat te laten zien is: dingen maken en delen. Geen gelikte productdemo's, geen hype, gewoon experimenten met een duidelijke gedachte erachter.

Als je hier iets ziet dat resoneert — herkenning, ergernis, een idee — laat het me weten via [LinkedIn](https://www.linkedin.com/in/thijsleufkens/) of [datawijs.nl](https://datawijs.nl).
